# SpeakAI

SpeakAI is an Enterprise-grade AI voice analysis and speech coaching platform. Analyze emotion, pacing, and clarity with professional insights.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
